## 📌 What is Nexus?

Nexus is a free, AI-powered web application built specifically for freelancers in **Pakistan, India, and the Arab world.**

Most freelance tools are expensive, Western-focused, and have no Urdu or local currency support. Nexus solves all of that — in one place, for free.

## 📱 Install as Mobile App (PWA)

Nexus is a **Progressive Web App (PWA)** — no App Store needed!

**Android (Chrome):**
1. Open the link
2. Tap 3 dots menu (⋮)
3. "Add to Home Screen"
4. Done! ✅

**iPhone (Safari):**
1. Open the link
2. Tap Share button (□↑)
3. "Add to Home Screen"
4. Done! ✅

**Laptop (Chrome):**
1. Open the link
2. Click download icon in address bar
3. Install
4. Done! ✅
